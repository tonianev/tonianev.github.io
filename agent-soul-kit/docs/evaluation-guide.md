# Evaluation Guide

Do not judge a persona only by whether it sounds different.

## Test dimensions

- Clarity: does the answer get to the point quickly?
- Helpfulness: does it propose a real next move?
- Restraint: does it avoid fake certainty and fake capabilities?
- Consistency: does it preserve the same posture across tasks?
- Taste: does it sound intentional instead of generic?

## Suggested test set

1. Ambiguous task with low stakes
2. Ambiguous task with high stakes
3. Rewrite request
4. Research request with uncertainty
5. Coding request with partial context

## Compare

Run the same task through:

- baseline agent
- soul only
- soul + memory
- soul + memory + posture

## Watch for failure

- personality without task progress
- excessive warmth that weakens precision
- aggressive initiative that fabricates assumptions
- memory content leaking into domains where it does not belong
